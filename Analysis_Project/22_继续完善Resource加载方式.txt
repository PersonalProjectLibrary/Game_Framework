3、更新CloseWnd(...)
（1）方法修改
将
CloseWnd(string name, bool destory = false, bool resource = false)
{
    Window wnd = FindWndByName<Window>(name);
    CloseWnd(wnd, destory,resource);
}
CloseWnd(Window window, bool destory = false, bool resource = false){
    ...
   if (!resource){...}
    ...
}
改为：
public void CloseWnd(string name, bool destory = false)
{
    Window wnd = FindWndByName<Window>(name);
    CloseWnd(wnd, destory);
}
CloseWnd(Window window, bool destory = false, bool resource = false){
    ...
   if (!window.Resource){...}
    ...
}
这样就不影响CloseAllWnd()的使用
